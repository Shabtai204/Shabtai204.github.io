ARCHITECTURE

Click any component to expand details.

C++ Agent / Python Simulator
Windows monitoring agent — 12 concurrent Win32 threads
Twelve modules run as concurrent Win32 threads with event-based signaling. Captures screenshots via GDI BitBlt, keystrokes via low-level keyboard hook (WH_KEYBOARD_LL), webcam frames via OpenCV, 30 s audio clips via WinMM, Chrome/Edge SQLite history, IP geolocation via libcurl, and more.

A Python simulator mirrors all 12 modules for development — no Visual Studio or OpenCV installation required. Pre-seeded test devices are ready to use out of the box.

Spring Boot 3 REST API
Java 17  •  PostgreSQL 14  •  Docker Compose
25+ endpoints handling device registration, data ingestion, user management, session auth, event auditing, and remote-action dispatch.

Spring Security 6 with PBKDF2+SHA-256 password hashing, 7-digit OTP 2FA via SMTP, Bucket4j rate limiting (150 req/min), IP lockout at 5 failed attempts, account lockout at 10 (10-min block), dictionary-attack prevention against 14 million leaked passwords, Spring AOP audit logging, paginated responses, and daily Excel report generation.

SECURITY MANIFEST

SECURITY_MANIFEST.v2 ─────────────────────────────
[✓] TLS 1.2 — Transport layer encryption for all agent ↔ server communication
[✓] AES-256 / CBC — Surveillance data encrypted at rest with a random IV per record
[✓] PBKDF2 + SHA-256 — Password storage with random salt via Spring Security encoder
[✓] OTP 2FA — 7-digit one-time code sent by email on every new registration
[✓] Brute-Force Lockout — IP blocked at 5 attempts, account at 10 — 10-minute cooldown
[✓] Rate Limiting — 150 requests / minute per client enforced by Bucket4j
[✓] Dictionary Attack Prevention — Passwords checked against 14 million leaked passwords (rockyou.dict)
[✓] SQL Injection Prevention — JPA prepared statements used throughout the data access layer
[✓] XSS Protection — X-XSS-Protection header + Content Security Policy on all responses
[✓] Session Management — 24-hour cookie timeout, RBAC with PLAYER / DEVICE roles
